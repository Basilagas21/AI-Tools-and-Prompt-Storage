# Python Web Scraper

### Prompt Details
- **Source:** Personal project
- **Goal:** Write a Python script to scrape product names and prices from an e-commerce website.
- **Notes:** Use the `requests` and `BeautifulSoup` libraries.

### Prompt Text
> Create a Python script that takes a URL of an e-commerce category page as input. The script should extract the name and price of each product listed on the page and save the data to a CSV file.
